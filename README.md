## Galaxy OS
This is a simple computer desktop that works in your browser. I made it to learn web development.

I used AI to help me fix problems and understand code when I got stuck.

The project took about 2 hours to make. It shows a fake operating system with a login screen and small apps. First you see a welcome message, then a loading bar. After that you press any key and type your name. The name must have between four and ten letters.

When you finish login, you see the desktop. There is a clock on top. There are five apps you can open. Settings lets you change dark or light mode. Calendar shows the month. Calculator does math. Tic Tac Toe is a game for two players. Translator changes your message with a secret code.

I learned many things from this project. I learned how to change things on the page with JavaScript. I learned how to move windows with the mouse.

Some parts were difficult. Making windows move was hard. The calendar with months was confusing. The tic tac toe game needed a lot of thinking.

Sometimes windows go off the screen. The calculator breaks if you type strange things. Background pictures don't save after refresh.

I want to make this better in the future. I could stop windows from going off screen. I could add more apps. I could make it work on phones too. But for now, I am happy with what I made.
